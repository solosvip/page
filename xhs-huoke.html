将获客助手配置在小红书使用

获客助手 × 小红书聚光：从建链接到投放的完整步骤

1.  创建获客链接，配置成员及欢迎语等

企业可以从企业微信「获客助手」应用首页，点击“创建获客链接”来创建新链接，每个获客链接可配置最多500名成员，客户点击链接后，将平均分配一名成员，当某名成员当日暂无法正常添加客户时，会在分配时自动跳过该成员，避免客户流失。创建链接时，还可配置欢迎语、标签等。

也可以通过 api 调用获客链接管理创建和编辑获客链接，如使用 api创建链接，可结合“欢迎语“及“标签”的能力，为获客链接配置专属欢迎语，或为通过链接添加的客户自动打上标签。

2.  将获客助手授权给平台使用

从小红书聚光平台，点击「工具」-「企微管理」，点击「前往企微授权」或者「企微扫码授权」。出现二维码后，请使用管理员的企业微信扫描此二维码，完成授权。如主账户未能在聚光平台上找到「企微管理」入口，或者子账户在投放创建计划时无「企微」相关推广目标，请联系小红书平台的广告销售申请权限。

在授权时，请选择将要在此平台使用的获客链接进行授权，选择后点击「同意以上授权」即可。请注意，授权给平台的链接将不可用于其他场景，专链专用才可保证数据清晰。下图为授权流程示意图，供参考。

后续如果希望增删授权给平台的链接，可从再次扫码进入授权页，以授权更多链接给平台或删除已授权的链接。也可从企业微信管理后台「客户联系」-「获客助手」-「授权接入获客助手」中，增删链接。

3.  创建带有获客链接的广告落地页

使用获客助手在小红书投放时，无需经过落地页，点击按钮后即可添加企业微信。因此，直接创建投放计划即可。

4.  创建投放计划，开启高效获客

创建投放计划时，投放目标选择「客资收集」，广告类型选择「搜索」，推广标的选择「笔记」；其他链路当前暂未开放，请务必按上述指引创建。推广目标推荐选择「添加企微量」（笔记胶囊/按钮点击量），为保证广告效果，根据小红书平台要求，必须开启智能拓词。

设置投放计划中的创意时，转化组件请选择「企微组件」；并在「组件位置」中，同时勾选互动栏胶囊和置顶评论。组件按钮文案可自定义。选择获客链接时，请选择已授权给平台的获客链接。置顶评论文案需明确指引用户点击左下角按钮；如：请点击笔记左下角按钮“立刻参与”。其余内容根据实际情况填写即可。

5.  查看数据，持续优化效果

投放后，可在企业微信「获客助手」-「获客链接」中查看获客链接相关数据，如链接打开人数、添加客户数等；若获客链接异常，无法添加客户时，将通过获客助手应用及API通知，修改成员配置后即可继续添加。

广告投放数据，可在聚光推广页面中查看「添加企微/成功添加企微量/企微开口量」数据和对应成本（数据将自动回传，无需手动配置）。
